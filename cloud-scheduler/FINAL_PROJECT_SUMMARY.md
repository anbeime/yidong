## 🚀 使用指南

### 快速开始
1. **启动演示平台**:
   ```bash
   cd cloud-scheduler
   python integrated_demo.py
   ```

2. **访问界面**: http://localhost:8084

3. **开始生成**:
   - 在界面中描述您的应用需求
   - 选择应用类型（电商、教育、CRM等）
   - 点击"开始生成应用"
   - 等待AI分析和代码生成
   - 下载完整的项目代码包
